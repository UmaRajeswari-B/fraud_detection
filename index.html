FRAUD DETECTION IN FINANCIAL TRANSACTION

Introduction

  • The increasing reliance on digital payments and online banking has made financial systems more vulnerable to fraudulent activities. Fraudulent transactions, if undetected, can cause severe monetary loss and erode customer trust. To address this challenge, a software model has been developed for Fraud Detection in Financial Transactions.

  • This model analyzes transaction data such as amount and time of transaction to identify unusual or high-risk patterns. For example, transactions above a certain threshold during odd hours are flagged as potentially fraudulent. Using adaptive learning, the system continuously refines its detection capability by recognizing both fraudulent and legitimate behaviors, thereby improving accuracy over time.

  • By classifying transactions into normal and fraudulent, this model helps financial institutions minimize risks, strengthen security, and provide customers with a safer digital transaction environment.

FRAUD DETECTION CODE

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
np.random.seed(42)
n = 500
amount = np.random.uniform(50, 10000, n)
time_of_day = np.random.randint(0, 24, n)
is_fraud = (amount > 5000) & ((time_of_day < 6) | (time_of_day > 21))
is_fraud = is_fraud.astype(int)
df = pd.DataFrame({
'Amount': amount,
'TimeOfDay': time_of_day,
'IsFraud': is_fraud
})
plt.figure(figsize=(10, 6))
ax = plt.gca()
ax.set_facecolor('#f8f8f8')
plt.scatter(df[df['IsFraud'] == 0]['TimeOfDay'], df[df['IsFraud'] == 0]['Amount'],
color='#2E8B57', s=60, alpha=0.6, edgecolors='black', label='Normal
Transaction')
plt.scatter(df[df['IsFraud'] == 1]['TimeOfDay'], df[df['IsFraud'] == 1]['Amount'],
color='#DC143C', s=100, alpha=0.9, edgecolors='black', label='Fraudulent
Transaction')
plt.title('AI-EBPL-Fraud Detection in Financial Transactions', fontsize=15, weight='bold')
